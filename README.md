# E-commerce Sales Analysis

## Project Overview

This project analyzes e-commerce sales data using Python, SQL, SQLite, and Power BI.
The goal of the project was to clean raw transactional data, perform exploratory data analysis (EDA), create SQL-based business insights, and build an interactive dashboard.

---

## Technologies Used

- Python
- Pandas
- SQLite
- SQL
- Power BI
- Jupyter Notebook
- VS Code
